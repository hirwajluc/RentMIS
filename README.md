# RentMIS — Rental Management Information System

RentMIS is a production-grade web application built for the Rwandan rental market. It digitises the full lifecycle of property rental — from landlord and tenant registration through to contract signing, rent collection, and fiscal compliance — replacing fragmented paper-based processes with a secure, auditable digital platform.

The system supports four roles: **Admin**, **Landlord**, **Agent**, and **Tenant**, each with a tailored dashboard. It integrates natively with Rwanda's **NIDA** national identity system, the **Rwanda Revenue Authority (EBM/Inkomane)** fiscal receipting platform, and **GLSPay** for online payments.

Built with **Java 21** and **Spring Boot 3.2.4**, backed by **MySQL 8**, and deployable on any Linux server.
